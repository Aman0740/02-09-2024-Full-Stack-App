» Project Overview

This project is a web application that includes multiple pages like Home, About, Signup, and Login. The project is built using Node.js, Express.js, and MongoDB for the backend, with HTML and CSS for the frontend.

» Pages

  • Home: The main landing page of the application.
  • About: This page (you're here!), providing details about the project.
  • Signup: A page where users can register for an account.
  • Login: A page for existing users to log in.

» Features

  • User Authentication: Signup and login functionality.
  • Responsive Design: The application is mobile-friendly and works across devices.
  • MongoDB Integration: All user data is stored and managed in a MongoDB database.
